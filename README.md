# Sistema de Inventario de Productos

## Descripción

Aplicación web desarrollada para la Evaluación Sumativa 2 de Programación Front End.

Permite:

- Agregar productos.
- Buscar productos.
- Eliminar productos.
- Mostrar estadísticas.
- Guardar información mediante LocalStorage.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Validación de formularios
- Manipulación del DOM
- Arreglos y Objetos
- Expresiones regulares
- Seguridad utilizando textContent y createElement
- LocalStorage
