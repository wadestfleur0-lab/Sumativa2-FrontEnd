# Uso de Inteligencia Artificial

Para el desarrollo de esta aplicación se utilizó ChatGPT como apoyo para:

- Crear la estructura HTML.
- Diseñar la interfaz con CSS.
- Organizar el código JavaScript.
- Implementar validaciones mediante expresiones regulares.
- Aplicar buenas prácticas de seguridad evitando el uso de innerHTML.
- Modularizar funciones para mejorar la legibilidad del código.

Las sugerencias fueron revisadas y adaptadas antes de incorporarlas al proyecto.
